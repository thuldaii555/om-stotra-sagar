## Phase 2 UX Overhaul

Phase 2 restores and verifies Admin access, removes redundant Browse Library CTAs, and rebuilds the main user experience around a calmer manuscript-inspired home screen, slim global navigation, compact Library list, polished reader panel, tighter spacing, and a simplified footer. The work keeps the local-first CMS flow intact while improving scanability, reading comfort, dark mode access, and overall product focus.

# Future Backend Notes

Om Stotra Sagar v1 is intentionally local-first and does not depend on Firebase, login, or any backend service.

This file keeps the prior backend ideas in one place so the v1 documentation stays focused.

Possible future additions, if the product ever grows beyond the local-first baseline:
- Firebase Auth for optional sign-in
- Firestore for synced content
- Nepali language support
- Live Panchang calculation service
- Admin custom claims for remote content moderation

None of those are required for the current v1 release.
